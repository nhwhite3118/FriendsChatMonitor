# Friends Chat Monitor

Runelite plugin which sends FC messages to a 3rd party server for de-duplication, which then sends the unique messages to Discord. As long as at least one user with the plugin is in FC, unique messages will be forwarded to Discord.

The 3rd party server de-duplicating the messages costs me money to run so I will be charging $5/mo for unlimited messages. There will be a free trial tier with limited messages per day.
